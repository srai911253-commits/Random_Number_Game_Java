Random Number Game in Java

This is a simple Java project where the user guesses a randomly generated number between 1 and 100.
Features
*Random number generation
*User input using Scanner
*Looping with do-while
*Conditional statements(if-else)

Technologies Used
*Java

 How to Run
 1.Compile the program: javac Random_Number.java
 2.Run the program: java Random_Number
